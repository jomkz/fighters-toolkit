# In-Game Menu Layout (.MNU)

FA_2.LIB contains 12 `.MNU` files. These define the top-level structure of the FA in-game menus — the screens presented between missions for mission selection, loadout, and campaign management. Individual dialog boxes within menus are defined by `.DLG` files.
