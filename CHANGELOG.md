## [0.1.0] - 2026-05-16

### Added
- Initial release of fighters-toolkit
- `ft_lib` — C++17 static library for reading and writing Jane's Fighters Anthology asset formats (LIB, PIC, SEQ, BRF/OT/NT, audio, mission, SH, CB8, RAW)
- `ft` — command-line tool for unpacking, inspecting, and repacking FA assets
- `ft-gui` — ImGui/DirectX 11 GUI editor for FA LIB archives with three-panel layout
